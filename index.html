HTML

  • The head element contains metadata for the webpage. It is not seen on the active webpage.
  • The body element is visible to the user.
  • HTML provides the body of the webpage.

CSS

  • Margin represents how much space is around the outside of an element.
  • Padding represents how much space is around the content inside an element.
  • CSS provides the style of the webpage.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git switch branch-name: switches to another branch
    • git add -A: stages files in the current branch
    • git commit -m “updated project file”: Saves the changes made to the branch
    • git pull origin main: make sure latest version of the "main" branch exists
    • git push origin feature/project: send file to remote repo

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JS provides the functionality of the webpage.
